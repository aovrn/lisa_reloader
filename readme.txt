Данное приложение может использоваться для перезапуска стандартного шелла на магнитолах kia/hyundai
Для подмены окна и перехвата сообщений.
Для анализа активности приложений в wince.
И много для чего еще, поведение аолностью настраивается файлом конфигурации.

В папке conf примеры конфигов, имя конфига должно быть таким же как имя приложения, но с расширением .cfg
Конфиг состоит 
* из параметров приложения 
* секций выполнения
	Секции выполняются последовательно.
	Действия внутри каждой секции неразрывны (кроме PostWnd с timeout большим 0)
	Следует иметь в виду, что между секциями может прийти сообщение, которое поменяет порядок выполнения.

Управление:
* Окно можно перетаскивать
* Тройной клик -- убрать сплеш-скрин
* Клик -- листать лог
* Повторный тройной клик -- выйти
* Повторный запуск приложения с таким же классом и заголовком -- сделать видимым предыдущее окно, если оно было скрыто и завершить работу.
