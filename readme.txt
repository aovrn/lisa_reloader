Данное приложение используется для перезапуска стандартного шелла на магнитолах kia/hyundai
Также оно может использоваться для анализа активности приложений в wince
В папке conf примеры конфигов, имя конфига должно быть как и имя приложения с расширением .cfg

Управление:
* Окно можно перетаскивать
* Тройной клик -- убрать сплеш-скрин
* Клик -- листать лог
* Повторный тройной клик -- выйти
